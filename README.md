Streamlit Magnetic Anomaly Visualization

Deskripsi

Aplikasi ini merupakan aplikasi web berbasis Streamlit untuk visualisasi data anomali magnetik, yang menampilkan:

Anomali magnetik observasi

Anomali magnetik regional

Anomali magnetik residual


Visualisasi ditampilkan dalam bentuk peta kontur (contour map) yang interaktif, dilengkapi dengan pengaturan colormap, skala otomatis/manual, serta fitur unduh hasil visualisasi.

Aplikasi ini telah dikemas menggunakan Docker dan disiapkan untuk diunggah ke Google Container Registry (GCR).


---

Fitur Utama

Visualisasi 3 jenis anomali magnetik (Observasi, Regional, Residual)

Pilihan colormap (cmap) interaktif

Mode Auto Scale dan Manual Scale (vmin & vmax)

Tampilan multi-kolom

Ekspor hasil visualisasi ke format PNG

Siap dijalankan secara lokal maupun melalui container
